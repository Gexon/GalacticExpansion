**GalacticExpansion (GLEX)** — server-side DLL мод для Empyrion: Galactic Survival (v1.15 Experimental) для имитации «живой» захваченной планеты и дальнейшей экспансии Zirax.

## Цель
Создать систему, в которой Zirax:
- встречают игрока патрулями при высадке;
- строят базу по стадиям “площадка → база → улучшения”;
- «добывают ресурсы» (симуляционно) и расширяются;
- атакуют и мешают высадке игроков;
- при уничтожении объектов игроками реагируют (усиление, откат стадии, повторное строительство).

## Что реально контролируем через публичный API
- Спавн/удаление/телепорт структур и сущностей: `Request_Entity_Spawn`, `Request_Entity_Destroy*`, `Request_Entity_Teleport`, `Request_Entity_ChangePlayfield` и получение позиции `Request_Entity_PosAndRot`.
- Списки сущностей/структур: `Request_Playfield_Entity_List`, `Request_GlobalStructure_List`, реакции на события `Event_Player_ChangedPlayfield` и т.п.
- Триггер “готовых” AI-действий (дроны/волны/дронбаза) через консольный AIM (`aim ...`) по `Request_ConsoleCommand`.

## MVP (первый релиз)
- 1 “материнская” планета, 1 Zirax база (стадийная).
- Патрули/охранники/строители вокруг этапов строительства.
- “Логистический” корабль: видимый AI-вессел появляется над планетой, “доставляет” капсулу, исчезает.
- Конфиг лимитов: максимум баз = 1 на планету, лимиты NPC/дронов/кораблей, интервалы атак.
- Реакция на разрушение: если игрок ломает стадию — откат/перестройка.
