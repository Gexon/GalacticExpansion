# GalacticExpansion (GLEX)

**GalacticExpansion (GLEX)** — server-side DLL мод для Empyrion: Galactic Survival, который имитирует «живую» захваченную планету и дальнейшую экспансию Zirax: патрули при высадке, стадийное строительство базы, симуляцию добычи/логистики и реакции на действия игроков.
