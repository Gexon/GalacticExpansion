# Модуль `AimOrchestrator`

**Ответственность**: безопасно вызывать AIM/консольные команды через `Request_ConsoleCommand` (whitelist + rate limit).  
**Аудитория**: разработчики.  
**Статус**: draft.  
**Связанные документы**: [`../06_security_model.md`](../06_security_model.md), [`../../config/ConfigReference.md`](../../config/ConfigReference.md).  

## Входы

- Запросы от доменных модулей: “создай патруль”, “запусти волну”, “усиль оборону”.

## Выходы

- `Request_ConsoleCommand` с командами вида `aim ...` (и только разрешёнными).

## Инварианты

- **Запрещено** отправлять произвольные команды: только whitelist.
- Любая команда проходит проверку лимита `MaxAIMCommandsPerMinute`.
- Все отклонения (запрещено/лимит) логируются.

## Edge cases

- Спам событий (например, много разрушений подряд): AimOrchestrator должен деградировать “мягко” — отклонять излишки, а не лагать сервер.
