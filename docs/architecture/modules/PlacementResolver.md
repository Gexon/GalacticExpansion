# Модуль `PlacementResolver`

**Ответственность**: выбрать координаты для спавна сущностей (корабль, стройплощадка, аутпост) с учётом ограничений и фоллбеков.  
**Аудитория**: разработчики.  
**Статус**: draft.  
**Связанные документы**: [`../05_placement_and_spawn.md`](../05_placement_and_spawn.md).  

## Входы

- Цель размещения (тип: корабль/BA/аутпост, желаемый “центр”).
- Контекст playfield (если доступен), наличие игроков.
- Ограничения из конфига (радиусы, запретные зоны, `DropShipSpawnAltitude`).

## Выходы

- Координаты/ориентация для `EntitySpawnInfo`.
- Причина отказа/перевыбора, если точка невалидна.

## Инварианты

- Никогда не возвращаем “случайную” точку без логики: любая точка должна быть воспроизводима/объяснима (лог).
- Если API не даёт “поверхность” — используем эвристику + фоллбек.
